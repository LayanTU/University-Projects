## 📁 My Projects

### 1. Java Bank System
Located in `/FlowerBank-Java`
* **Focus:** Object-Oriented Programming (OOP)
* **Description:** A console-based banking application built in Java. It allows users to manage their accounts, check balances, and process deposits or withdrawals. I used core OOP concepts like classes and inheritance to separate different types of users (like Standard vs. Platinum accounts).

### 2. Company Database
Located in `/Company-Database-SQL`
* **Focus:** SQL & Relational Databases
* **Description:** A database built for a fictional company. I wrote the SQL queries to create multiple tables for employees, departments, and projects. I also set up the primary and foreign keys to make sure all the data connects together properly without errors.

### 3. Healthy Habits Website
Located in `/HealthyHabits-Web`
* **Focus:** HTML & CSS
* **Description:** A multi-page website focused on healthy living. I built the layout and design from scratch to practice structuring web pages, linking files, and making the site look clean and readable.

### 4. Taif University Classroom Recreation
* **Located in:** `/Classroom-OpenGL`
* **Focus:** C++ & OpenGL (2D/3D Computer Graphics)
* **Description:** An interactive 2D/3D scene recreating a university classroom. I utilized OpenGL primitives and matrix transformations to build the environment, and implemented state machine management to handle dynamic lighting, transparency blending and 2D character animations within a 3D space.
